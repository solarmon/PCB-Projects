# Amiga A500 Keyboard Connector Breakout

This is a breakout board for the A500/A500+ keyboard connector to allow access to the signals on it.

The keyboard connector has the following pinouts and signals:

1. _KBCLOCK
2. _KBDATA
3. _KBRESET
4. +5V
5. (KEY)
6. GND
7. STATUS (POWER LED)
8. INUSE (DRIVE LED)
